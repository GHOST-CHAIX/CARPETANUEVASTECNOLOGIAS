Todos los formularios web, manejan metodos para el envio de las diferentes peticiones que realiza el usuario. Debemos recordar que la mayoria de las aplicaciones en web trabajan sobre un arquitectura cliente servidor significa que el cliente realiza una peticion (request) y el servidor debe de atender mediante una respuesta (response). Los metodos mas comunes son:

GET:

Este metodo se encarga de enviar la informacion por la parte estandar del navegador mediante la url estandar. Propiciando inyecciones de codigo generando violaciones al sistema

POST:

Este medoto se encarga de enviar la informacion de igual manera por el metodo estandar. Siendo no visible para el usuario. Ofreciendo una mayor seguiridad

GET solo se usa para operaciones de lectura

POST: Para la entrada de datos a nivel servidor

DELETE: Sirve para eliminar un recurso en especifico

PUT: Almacena la informacion de un recurso para su actualizacion
